# ystack Configuration Templates ⚡

This directory contains the core configuration files for your multi-agent development swarm. 
When you're ready to use ystack in a new project, follow these steps:

---

## ── Setup 🚀

1.  **Read `AGENTS.global.md`**: These are the shared rules that will apply to all your projects.
2.  **Add `AGENTS.project-template.md`**: Use this to define the technology stack, project goals, and specific rules for your new project.
3.  **Run `bootstrap.md`**: Paste the final prompt in this file into your **Antigravity Agent Manager** — it will explain to the AI how to behave.

---

## ── Files 📂

- **`AGENTS.global.md`**: Global operating principles (e.g. read before write, simple over clever).
- **`AGENTS.project-template.md`**: Template for project-specific details.
- **`bootstrap.md`**: The initial seed prompt to feed your AI orchestrator.
