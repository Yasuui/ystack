# MCP Configuration — ystack ⚡

> **Local Model Context Protocol (MCP) server settings for ystack.**

This directory contains the configuration for the Model Context Protocol (MCP) servers used by the swarm to interact with the outside world.

---

## ── Servers 🔌

- **GitHub**: Handles all git operations, PRs, and issue tracking.
- **Context7**: Real-time programming library intelligence.
- **Playwright**: Browser automation and E2E testing.
- **Supabase**: Backend database and authentication management.
- **FireCrawl**: Deep web research and extraction.

---

## ── Customizing Settings ⚙️

You can modify the `settings.json` file in this directory to add or remove MCP servers per project.
