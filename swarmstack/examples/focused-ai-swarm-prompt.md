## ── Swarm Coordination Script 🧬

### Orchestrator Instruction:
Analyze the `Focused AI` architecture and delegate to specialists.

1.  **Frontend Agent**:
    - Build a system tray icon with a calm dark overlay.
    - Implement a `glassmorphism` card for the meeting summary.
    - Use `@taste-soft-skill` for premium UI.

2.  **Backend Agent**:
    - Integrate `faster-whisper` for local transcription.
    - Setup `Silero VAD` for silence filtering.
    - Create a Python / FastAPI bridge for the `Gemini 2.0 Flash` API.

3.  **QA Agent**:
    - Verify that the audio stream is correctly captured and transcribed.
    - Run integrity tests on the `SQLite` storage.
    - Test for memory leaks during long meetings.

4.  **Ship Agent**:
    - Bundle the Tauri V2 / Next.js app.
    - Deploy the initial v0.1 release to GitHub.
